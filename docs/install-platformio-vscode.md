# Install PlatformIO In VS Code

If you are not sure whether `PlatformIO` is installed:

1. Open `VS Code`
2. Press `Ctrl+Shift+X`
3. Search for `PlatformIO IDE`
4. Install the extension published by `PlatformIO`
5. Restart VS Code
6. Re-open the folder `C:\Users\Kosmo\Documents\Playground\lora20-firmware`

You should then see:

- PlatformIO icon in the left sidebar
- environment `heltec_v4_bootstrap`
- `Build`, `Upload`, and `Monitor` actions

If the extension is already installed, opening this folder should trigger dependency download automatically.
